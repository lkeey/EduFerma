# Platform Roadmap

- Parent portal: schedule, progress and reports without editing assignments.
- Notifications: homework reminders, upcoming lesson reminders, pending review for teachers.
- Hints: per-task hints and hint usage logs.
- Motivation: streaks and consistency badges without turning learning into a game.
- Lesson notes: after-lesson notes that can adjust the plan.
- File attachments: task files and uploaded solutions through Vercel Blob or compatible storage.
- Import diagnostics: imported, needs-review and invalid task counts.
- Audit log: plan edits, assignment publishing and review actions.
- Consent: public testimonials/results only with granted consent.
- AI later: recommendations, homework generation and mistake analysis after core flows are reliable.
