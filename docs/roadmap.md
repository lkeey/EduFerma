# Roadmap

## MVP

- Russian landing with Telegram CTA.
- Invite-only student and teacher dashboards.
- Demo seed data.
- Dry-run task bank import.
- Drizzle + Neon schema.
- Preview deploy on Vercel.

## Later

- Real scheduler integration.
- File uploads through Vercel Blob.
- Background worker jobs.
- Deeper analytics and mastery rollups.
- Billing, PWA, calendar and code execution only after separate product review.
